# LogPose AI 🧭

![LogPose AI](https://img.shields.io/badge/Status-Active-success) ![License](https://img.shields.io/badge/License-MIT-blue) ![Angular](https://img.shields.io/badge/Angular-19.0.0-dd0031?logo=angular) ![FastAPI](https://img.shields.io/badge/FastAPI-0.104.1-009688?logo=fastapi)

LogPose AI is a production-grade, multi-agent AI orchestration platform inspired by modern AI coding assistants (like Proxima, Devin, and Cursor), but built entirely from scratch with maximum scalability, clean architecture, and extensibility in mind.

It uses a stateful agent graph to break down complex coding tasks, plan architectures, write production code, and review it—all while streaming its thoughts and outputs in real-time to a beautiful browser-based IDE.

## ✨ Features

- **🧠 Multi-Agent State Graph (LangGraph)**
  Tasks are seamlessly routed through specialized agents: **Planner → Architect → Coder → Reviewer**.
- **⚡ Real-Time WebSocket Streaming**
  Watch the agents "think" and generate code in real-time. No waiting for long HTTP polling requests.
- **🔑 Bring-Your-Own-Key (BYOK) Dynamic Routing & Failover**
  Supports Google Gemini, OpenAI, Anthropic, Groq, and local **Ollama** instances dynamically. The smart LLM router automatically fails over to available keys if a preferred provider isn't configured, preventing crashes and hangs.
- **💻 Authentic VS Code Experience (Monaco Editor)**
  Interact with generated files using a fully integrated Monaco Editor with auto-detected syntax highlighting, dynamic height mapping, and dark mode themes.
- **🛡️ Bulletproof Workspace Parser**
  A rigorous regex-driven Markdown parsing engine that flawlessly extracts code blocks generated by the LLM while safely ignoring code found within conversational shell commands.
- **🖥️ Live Workspace Terminal**
  Execute scripts (Python, Node, etc.) and view console outputs directly within the browser interface safely sandboxed in session workspaces.
- **🎨 Stunning Modern UI**
  Built with Angular 19 and Tailwind CSS, featuring glassmorphism, dark mode, dynamic Signal states, and responsive layouts.

## 🏗️ Architecture

- **Backend:** FastAPI + LangGraph + LangChain + Uvicorn
- **Frontend:** Angular 19 + RxJS + Tailwind CSS + ngx-monaco-editor-v2
- **Communication:** REST APIs (State Management & Execution) & WebSockets (Token Streaming)
- **State Management:** Angular Signals for highly optimized, zone-less reactivity.

## 🚀 Getting Started

### Prerequisites
- Node.js (v18+)
- Python (v3.10+)
- npm or yarn

### 1. Backend Setup

Navigate to the `backend` directory and set up a virtual environment:
```bash
cd backend
python -m venv venv
source venv/bin/activate  # On Windows use: venv\Scripts\activate
```

Install the dependencies:
```bash
pip install -r requirements.txt
```

Run the FastAPI server:
```bash
uvicorn app.main:app --reload --port 8000
```
*The backend will be running at `http://localhost:8000`*

### 2. Frontend Setup

Navigate to the `frontend` directory:
```bash
cd frontend
```

Install the dependencies:
```bash
npm install
```

Run the Angular development server:
```bash
npm start
```
*The frontend will be running at `http://localhost:4200`*

## 💡 Usage

1. Open `http://localhost:4200` in your browser.
2. Click **⚙️ Settings (Keys)** in the bottom left corner to add your preferred LLM API key (e.g., Gemini, OpenAI, Anthropic).
3. Create a new Session from the top-left **`+`** button to initialize a fresh, isolated workspace.
4. Provide an instruction to the orchestrator (e.g., *"Build a Python script that fetches the current price of Bitcoin from the CoinGecko public API..."*).
5. Watch the Multi-Agent pipeline execute the request.
6. Explore generated files in the Workspace, edit them using the Monaco Editor, and execute them via the Live Terminal!
